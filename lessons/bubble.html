HKDSE Key Concepts

Algorithm Characteristics

  • Type: Exchange Sort.
  • Mechanism: Repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order.
  • Data Movement: Heavy. Lots of writing to memory.
  • Stability: Stable (Equal elements retain their relative order).

Time Complexity

  • Worst Case: O(n²)
    Array in reverse order.
  • Average Case: O(n²)
  • Best Case: O(n)
    Array already sorted (requires the 'swapped' flag optimization).
